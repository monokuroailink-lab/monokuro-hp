# モノクロ。 HP制作プロジェクト コンテキスト

> このドキュメントは HP 制作の方針書。ブランド本体の正典は `06_Guidelines_マニュアル/Core_Context.md` を参照すること。

---

## 1. プロジェクト概要

- **目的**: モノクロ。公式 HP の制作・運用
- **タグライン**: **想いを、伝わるかたちに。**
- **ターゲット**: 中小・個人事業主・スモールチームのうち「もったいない才能を持つ規格外の挑戦者」
- **対象外読者**: 権威性を求める大手BtoB層

### 立ち位置(三層)

| 層 | 像 |
|---|---|
| 業務 | コンサル(戦略・実装・運用伴走) |
| 人格 | 軍師ウラカタ(主役を勝たせる影) |
| メソッド | エンジニア比喩(売れない=バグ → 売れる=システム) |

---

## 2. サイトマップ

中小の挑戦者が「読んで→腑に落ちて→相談する」までを最短化する5ページ構成。

### [HOME] `index.html`
- ファーストビュー(タグライン + キーコピー + CTA)
- VISION / MESSAGE(軍師ウラカタの定義 → ABOUT 誘導)
- WORKS(代表実績数点)
- SERVICE(提供サービス6項目のサマリー)
- PROCESS(モノクロ・アルゴリズム4工程)
- NEWS(活動記録)
- COMPANY(会社情報)
- CONTACT CTA

### [ABOUT] `about.html`
- Concept / Philosophy(ウラカタの美学・三層アイデンティティ)
- Mission / Vision / Value
- 行動プロトコル(OVERRIDE_01〜FINAL OVERRIDE)
- Company Info

### [SERVICE] `services.html`
- 01 戦略コンサル / 構造設計
- 02 言語化 / コンセプト設計
- 03 LP / Web 制作
- 04 LINE / SNS / 広告運用
- 05 動画 / クリエイティブ制作
- 06 AI 運用支援
- ※ 単発ではなく「構造化 → 実装 → 運用」を地続きで提供する旨を明記

### [WORKS] `works.html`
- 実績ギャラリー(グリッド)
- カテゴリ絞り込み(LP / Web / 動画 / SNS運用 / AI運用 等)

### [CONTACT] `contact.html`
- 問い合わせフォーム(中小の挑戦者向けに敷居を下げた文言)
- 資料DLリンク

## 3. デザイン方針・トンマナ

詳細は Core_Context.md §8 ブランド原則を参照。HP特有の補足のみ記載。

### 文体
- **基本は詩的**(短句・余白・断定・象徴語)
- 詩的フレーズの直下に必ず**具体物**(サービス名・数値・工程)を置く
- バランス指針: **「詩で引き寄せ、論理で約束する」**

### 避けるコピー(HP既存表現の整理対象)
- ❌ `STRATEGY & VERBALIZATION FIRM >> MONOKURO.` のような英語ファーム調コピー
- ❌ 「BtoB向けクリエイティブ/デザインファーム」と自称する表現
- ❌ 大手BtoB風の権威付け
- ❌ 抽象3項目(戦略 / 言語化 / 実務伴走)で終わる Service 表記 → 具体6項目に展開

### 推奨表現
- ✅ 「想いを、伝わるかたちに。」(FV)
- ✅ 「自分は白、顧客は色。」
- ✅ 「売れない理由を、構造に変える。」
- ✅ エンジニア比喩(デバッグ / プロトコル / デプロイ / チューニング)

### ビジュアル
- モノトーン基調(黒・白・グレー)
- 「。」のミニマルな視覚アクセント
- 余白を取った抑制的レイアウト
- 詩的コピーは左寄せ・短句・改行多め
