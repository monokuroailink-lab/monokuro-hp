# モノクロ。 HP制作プロジェクト コンテキスト

## 1. プロジェクト概要
*   **目的**: 所属チーム「monokuro」の公式ホームページ作成
*   **コンセプト（理念）**: 
    *   **「想いを、伝わるかたちに。」**: 内側の熱や才能を、構造と言語と発信で「売れる仕組み」に翻訳する。
    *   **「自分は白、顧客は色。」**: 黒衣（ウラカタ）に徹し、主役に光を当てるアプローチ。
    *   **「現場と戦略の二刀流」**: 机上の空論でも精神論でもなく、構造の理性と現場の泥臭さの両方を持つ。
*   **ウラカタの美学（行動指針）**:
    *   光を奪わずに、光をつくる。
    *   失敗は自責、成功は相手の功績。
    *   誰よりも“見えている”存在になる。
    *   静かに燃える（温度感で挑戦者の背中を押す）。
    *   誰より泥をかぶって、誰より嬉し涙を流す。

## 2. サイトマップ（ページ構成案）
BtoB向けのクリエイティブ/デザインファームとして、信頼感と実績をシンプルかつ美しく伝えるための5ページ構成。

*   **[HOME] トップページ (`index.html`)**
    *   ファーストビュー（洗練されたタイポグラフィや動画）
    *   VISION/MESSAGE（Aboutへの誘導）
    *   PICK UP WORKS（最新/代表的な実績数点）
    *   SERVICES（事業内容のサマリー）
    *   NEWS（お知らせ）
    *   CONTACT CTA（お問い合わせへの導線）
*   **[ABOUT] 私たちについて (`about.html`)**
    *   Concept / Philosophy（チームの思想や世界観）
    *   Profile / Message（メッセージ、スタンス）
    *   Member（メンバー紹介、それぞれの強み）
    *   Company Info（チーム概要/会社概要）
*   **[SERVICES] 事業内容 (`services.html`)**
    *   01. Web & UI Design（Web制作、LP、UI/UXデザイン）
    *   02. Branding Strategy（コンセプト構築、ロゴ、CI/VI設計）
    *   03. Digital Marketing（広告、SNS運用、データ分析）
*   **[WORKS] 制作実績 (`works.html`)**
    *   実績一覧ギャラリー（グリッド表示）
    *   カテゴリ別絞り込み（Web, UI/UX, Brandingなど）
*   **[CONTACT] お問い合わせ (`contact.html`)**
    *   BtoB向けお問い合わせフォーム
