# モノクロ。HP 中身づくり 質問票

> 目的：HPのビジュアルは完成したが「結局モノクロって何屋なん？」が伝わる中身（コピー・サービス定義・対象・実績・人物像）を埋めるための事前ヒアリング。
> 使い方：各設問に箇条書きで答える。完璧な文章でなくてOK。**「言語化できへん」もそのまま書く**（そこが詰める論点になる）。

---

## SECTION 1｜事業の核 ― そもそも何屋さんか

1. **一言で言うと、モノクロは何屋ですか？**
   - 例：「BtoB向けのブランディング会社」「Web制作会社」「マーケティング支援」など
   - 複数あるなら**収益比率の高い順**に書く。

2. **30秒で初対面の人に説明するなら？**（エレベーターピッチ）

3. **「うちは○○屋ではない」と明確に言えるものは？**（やらないこと／断る仕事）

4. **売上の柱は今、何ですか？**（受託制作／顧問・伴走／自社プロダクト／コンテンツ／その他）

5. **半年〜1年後、どの柱を伸ばしたい？**（HPはこの未来像から逆算する）

---

## SECTION 2｜誰のために存在するか ― ターゲット

6. **理想のクライアント像（ペルソナ）**
   - 業種／規模（社員数・年商）／役職（決裁者は誰？）／所在地
   - 抱えてる悩み・課題

7. **このHPを「見てほしい一人」を具体的に挙げるなら誰？**（既存の取引先1社を思い浮かべて書くのがおすすめ）

8. **逆に来てほしくない問い合わせは？**（予算感／業種／温度感など）

9. **想定する最低取引単価／レンジは？**（HPの空気感に直結する）

---

## SECTION 3｜サービス ― 何を売っているか

現サイトでは `Web & UI Design` / `Branding Strategy` / `Digital Marketing` の3本立てだが、実態と合ってる？

10. **各サービスについて以下を埋めてください**
    | 項目 | Web & UI Design | Branding Strategy | Digital Marketing |
    |---|---|---|---|
    | 具体の成果物 |  |  |  |
    | 標準の納期 |  |  |  |
    | 価格レンジ（or 非公開でOK） |  |  |  |
    | 関わり方（受託／伴走／顧問） |  |  |  |
    | 直近の代表事例 |  |  |  |

11. **3本立てに違和感ある？** 増やす／減らす／統合したい区分は？

12. **「依頼の入口」になりやすいサービスはどれ？**（フロント商品）
    そこから何にアップセル・派生していく？

---

## SECTION 4｜なぜ自分たちか ― 差別化・強み

13. **競合・比較対象に挙がるのは？**（社名 or タイプ）

14. **その中でモノクロを選ぶ理由を、過去のクライアントは何と言っていた？**
    （実際の言葉そのまま引用がベスト）

15. **「ウラカタの美学」「現場と戦略の二刀流」は理念として強い。これを“買う側のメリット”に翻訳すると？**
    例：理念「自分は白、顧客は色」→ メリット「クライアントの色を消さない、過剰な“制作会社らしさ”を出さない仕上がり」

16. **数値で語れる強みは？**（継続率／リピート率／改善実績の数字／受賞歴／メディア掲載）

---

## SECTION 5｜実績 ― WORKSに何を載せるか

17. **公開許諾が取れている案件**を全部リストアップ
    - クライアント名／業種／関わった範囲／期間／成果（数値）／使える画像の有無

18. **公開NGだが匿名なら載せられる**案件は？（業種＋成果だけでも掲載価値あり）

19. **「これ載ってないと“何屋か”伝わらん」という看板事例は？**（最低3件）

20. **載せたいけど成果物が地味で映えへん案件**ある？ → ストーリー or プロセス図解で見せる方向に切り替えるか検討。

## SECTION 6｜チーム・人

21. **メンバー構成**（氏名／ロール／顔出しOK？／プロフィール公開範囲）

22. **代表 or 顔になる人物のストーリー**
    - これまでの経歴／なぜモノクロを始めたか／個人として一番得意なこと

23. **メンバー紹介に「強み」を載せるとして、各人の一言キャッチは？**
    （例：「事業の翻訳家」「現場の体温計」など）

24. **チーム名「モノクロ。」の由来は？**（句点まで含めた意味があるなら）

---

## SECTION 7｜ビジョン・ストーリー

25. **モノクロは何のために存在する？**（ミッション）

26. **5年後、モノクロはどうなっていたい？**（ビジョン）

27. **「これだけは譲れない」という判断基準は？**（バリュー／行動指針）
    ※ 既存の「ウラカタの美学」5箇条は活かす前提で、追加・編集あれば。

28. **創業のきっかけ／原体験エピソード**（あればABOUTの軸になる）

---

## SECTION 8｜トーン＆ボイス

29. **HPで使う一人称は？**（私たち／僕ら／モノクロは／その他）

30. **語尾・空気感のイメージ**（断定的／柔らかい／関西弁混じり／英語多め／ etc）

31. **NGワード／使いたくない表現は？**（例：「クリエイティブ」「ソリューション」を多用しない、など）

32. **参考にしたい “言葉づかい” のサイト・人・本**

---

## SECTION 9｜会社情報・ファクト

33. 正式名称／屋号／法人 or 個人事業
34. 設立年月
35. 所在地（公開範囲：番地まで／市区町村まで／非公開）
36. 連絡先（公開用メール／電話の有無）
37. SNS／note／YouTube などのURL
38. 取引銀行・適格請求書登録番号など、BtoB先方が確認したい情報の公開可否

## SECTION 10｜CONTACT後のフロー

39. **問い合わせが来たら誰が／何時間以内に返信する運用？**
40. **初回は無料相談？ 有料？**（HPに明記するか）
41. **問い合わせフォームで必須にしたい項目は？**（予算・希望納期・参考サイトURL など）
42. **「資料ダウンロード」「事例集PDF」など、フォーム以外の入口を作る予定は？**

---

## SECTION 11｜運用・更新

43. NEWS（お知らせ）はどのくらいの頻度で更新できる？（無理なら出さない方がいい）
44. WORKSは納品ごとに追加できる体制ある？（追加フローを誰が握る？）
45. ブログ／コラムの予定は？（`04_Owned_Media_自社コンテンツ` との関係整理）

---

## SECTION 12｜判断に迷ってること（自由記述）

46. このHPで「正直まだ迷ってる／決めきれてない」ことを全部吐き出す欄
    - 例：サービス3本立てで本当にいいのか／代表の顔出すべきか／英語表記どこまで使うか／etc.
